# UmaOS Sound Theme

FreeDesktop-compatible sound theme for UmaOS.

## Sound events

| Event | File | Description |
|-------|------|-------------|
| desktop-login | dialog-login.oga | Upbeat fanfare (~2 sec) |
| desktop-logout | desktop-logout.oga | Gentle wind-down chime |
| message-new-instant | message-new-instant.oga | Soft bell notification |
| dialog-error | dialog-error.oga | Comical stumble sound |
| trash-empty | trash-empty.oga | Thud sound |
| audio-volume-change | audio-volume-change.oga | Subtle click |

## Status

Sound files are placeholders. Replace with original or CC-licensed audio.
All sounds MUST be original or CC-licensed (no game audio rips).
