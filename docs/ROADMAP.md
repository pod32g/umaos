# UmaOS Roadmap

## Phase 1: Buildable branded ISO (done)

- Arch releng profile overlay
- UmaOS naming and MOTD
- guided installer command (`umao-install`)
- VM test workflow

## Phase 2: Better installer UX (initial delivery done)

- Calamares packaging path with official-first and optional AUR fallback
- auto-launch installer in live session
- prefilled defaults (locale geoip, ext4 preset, systemd-boot target)

## Phase 3: Full visual identity (skeleton done)

- Plymouth boot splash
- SDDM login theme skeleton
- KDE color scheme + icon mapping skeleton
- branded wallpaper skeleton

## Phase 4: Release engineering

- reproducible ISO build pipeline (CI)
- signed package repository
- updater channel and changelog workflow

## Phase 5: User-facing polish

- official docs site
- troubleshooting wizard
- one-command post-install setup profiles (gaming/creator/laptop)
